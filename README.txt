To install the project: 

Run npm run install-dev to install the project

To run project:
Run npm run dev


Note:
- products get deleted from DB when they run out


Used technologies:
FE: 
- React
- Redux
- React-router
- Materializecss

BE: 
- Node
- Express
- Mongoose with MongoDB

- Passport (for logging in with google)
- Stripe (for card transaction) the test card is: 4242 4242 4242 4242 
Just go for the manual verification for the card (or back arrow)

For the expiration date use any date in the future (e.g.:02 / 20)
For the CVV anything can be used (e.g.: 123)
